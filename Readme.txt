Nama Program  : Hinomaru Oogi Crest
Penulis  : Rhio Adjie Fabian


******* Description *******
Program ini menampilkan crest dari lambang Akitsukuni yaitu Hinomaru Oogi. Ada tiga jenis crest yang dikelompokkan berdasarkan animasi 
yang dilakukan, yaitu
1. Scaling - Rotasi (Crest di tengah layar)
2. Translasi - Rotasi (Crest di atas dan bawah)


******* How To *******
1. Jalankan HinomaruOogi_rev2.exe.
2. Pilih menu yang ingin dijalankan sesuai angka yang terdapat pada menu.
3. Kembali ke menu utama dengan menekan keyboard pada command window untuk menutup window BGI (jika memilih menu no 1) dan menekan keyboard pada window BGI untuk menutup window BGI itu sendiri (jika memilih menu no 2 dan 3)
